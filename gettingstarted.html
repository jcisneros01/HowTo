Getting Started

Set Up Your Account

  1. To be able to use the API, you will need a Spotify account. To sign up for a free account, go to Spotify.
  2. After your account has been created, you'll need to go to My Applications.
  3. Log in to Spotify
  4. Create a new application
  5. Add this URL to Redirect URLs whitelist: http://localhost:8888/callback
  6. Save the application
  7. Make note of your client ID and Client Secret Key. You'll need them later on to complete the tutorial.
