API Overview

End Points

The API allows client applications to access music catalog content and user data through several endpoints at the base URL: https://api.spotify.com. Each endpoint has it's own unique path. For a list of the different available endpoints visit the Web API Endpoint Reference.

Requests

The API is a RESTful service that allows access via standard HTTPS requests such as GET and POST.

Responses

Data is returned as a JSON object. Visit the Web API Object Model for a overview of the objects returned by the endpoints.

Authentication

You don't need permission to access catalog content, but to access user data you will need permission. For more information, please visit the Web API Authorization Guide.
